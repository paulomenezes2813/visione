# Os 8 Pilares do Ecossistema Visione — Detalhamento

Detalhamento de cada pilar do ecossistema (ver [Como Será o Negócio](../o-negocio.md)). Cada documento segue o mesmo template:

1. **Em uma frase** — o que é o pilar
2. **Papel no ecossistema** — por que existe, o que entrega
3. **Escopo / o que inclui** — componentes concretos
4. **Como funciona** — conceitos e mecânica (quando há metodologia)
5. **Entregáveis e produtos** — o que se vende/entrega
6. **Conexões** — como se liga aos outros pilares
7. **Métricas de sucesso** — como saber que está funcionando
8. **Questões em aberto** — decisões pendentes
